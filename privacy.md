**Last updated:** February 15, 2026

This app is a personal analytics integration used by the account owner to view and summarize WHOOP data.

## What data is accessed

- Sleep metrics
- Recovery metrics (e.g., HRV, resting heart rate)
- Strain/activity metrics
- Profile/account identifiers required for API access

## How data is used

Data is used only to generate daily health summaries, trend insights, and coaching suggestions for the authorized user.

## Data sharing

No personal health data is sold. No personal health data is shared with third parties except services required to run this integration (hosting, storage, logging).

## Data retention

Data is retained only as needed to provide summaries and trend analysis. The user may request deletion at any time.

## Security

Reasonable safeguards are used to protect tokens and health data, including encrypted transport (HTTPS) and restricted access to credentials.

## Your choices

- You can revoke WHOOP access at any time from your WHOOP account/app settings.
- You can request deletion of stored data and credentials.
